# Platform Integrity Systems

This repository contains practical notes on platform integrity for social platforms.
The focus is on backend and product systems.

## What this repo covers

- Rate limiting 
- Behavioral risk signals
- Messaging safety constraints
- Integrity metrics 

## What this repo does not cover

- Content moderation taxonomies
- Policy language
- Community guidelines

## Design assumptions

- Most abuse is predictable.
- Most harm is incentivized by speed and scale.
- Friction works best when it is targeted and reversible.
- Metrics often hide more than they reveal.
